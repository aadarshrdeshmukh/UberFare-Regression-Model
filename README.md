# 🚖 Uber Fare Prediction & Dynamic Pricing Analytics

A machine learning project predicting Uber ride fares using XGBoost, with an interactive Streamlit dashboard.

## 📁 Project Structure

```
uber_project/
│
├── uber_fare_notebook.ipynb    ← Full Colab notebook (run this first)
├── app.py                      ← Streamlit dashboard
├── requirements.txt            ← Python dependencies
├── uber_fare_model.pkl         ← Saved XGBoost model (generated by notebook)
└── cleaned_sample.csv          ← Sample data for dashboard (generated by notebook)
```

## 🚀 Deployment Instructions

### Step 1: Run the Notebook
1. Open `uber_fare_notebook.ipynb` in Google Colab
2. Run all cells — this will generate:
   - `uber_fare_model.pkl`
   - `cleaned_sample.csv`
3. Download both files

### Step 2: Deploy to Streamlit Cloud

1. Create a GitHub repo (e.g. `uber-fare-dashboard`)
2. Upload these files to the repo:
   - `app.py`
   - `requirements.txt`
   - `uber_fare_model.pkl`
   - `cleaned_sample.csv`

3. Go to [https://streamlit.io/cloud](https://streamlit.io/cloud)
4. Click **"New app"** → Connect GitHub → Select repo → `app.py`
5. Click **Deploy** — you'll get a live public URL in ~2 minutes

### Alternative: Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Dashboard Features

| Tab | Contents |
|-----|----------|
| 💰 Fare Prediction | Live fare estimate with breakdown & surge indicator |
| 📊 Pricing Analytics | Fare distribution, distance vs fare, monthly revenue |
| ⏰ Surge Patterns | Hour×Weekday heatmap, peak vs off-peak comparison |
| 🤖 Model Insights | Feature importance, model comparison table, business insights |
